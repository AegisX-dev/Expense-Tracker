# 💰 Expense Tracker

a web-based expense tracker that helps users manage their daily expenses efficiently. It allows users to add, edit, and delete transactions, categorize their spending, and visualize financial insights through interactive charts.

## ✨ Features

- **Add, Edit, and Delete Transactions**  
- **Categorized Expense Tracking**  
- **Real-time Expense Summary**  
- **Interactive Charts using Chart.js**  
- **Filter Transactions by Category**  
- **Data Persistence with Local Storage**  
- **Responsive & User-Friendly Interface**  

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Libraries:** Chart.js (for visualizing expenses)
